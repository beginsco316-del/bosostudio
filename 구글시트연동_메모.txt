보소스튜디오 Google Sheets 연동 메모

1. Google 드라이브에서 새 Google 스프레드시트를 만듭니다.
2. 상단 메뉴에서 [확장 프로그램] - [Apps Script]를 누릅니다.
3. Apps Script 화면의 기존 코드를 모두 지웁니다.
4. 이 저장소에 같이 올린 google-apps-script.gs 파일 내용을 전체 복사해서 붙여넣습니다.
5. 저장합니다.
6. 오른쪽 위 [배포] - [새 배포]를 누릅니다.
7. 유형은 [웹 앱]으로 선택합니다.
8. 실행 사용자는 [나], 액세스 권한은 [모든 사용자]로 설정합니다.
9. 배포 후 나오는 웹 앱 URL을 복사합니다.
10. 보소스튜디오 앱에서 [연동/백업] 메뉴로 들어갑니다.
11. Apps Script 웹앱 URL 칸에 복사한 URL을 붙여넣고 [저장]을 누릅니다.
12. [시트로 보내기]를 누르면 Google Sheets에 Customers, Visits, Reservations 시트가 생성됩니다.

다른 컴퓨터에서 같은 데이터를 불러오려면:
1. 같은 보소스튜디오 앱 주소를 엽니다.
2. [연동/백업] 메뉴로 들어갑니다.
3. 같은 Apps Script 웹앱 URL을 넣고 [저장]합니다.
4. [시트에서 가져오기]를 누릅니다.

주의:
- Google Sheets에는 고객명, 전화번호, 예약 정보가 저장됩니다.
- GitHub 저장소가 공개(Public)이면 고객 백업 파일(initial-data.json)은 올리지 않는 것이 안전합니다.
- google-apps-script.gs 파일은 고객 데이터가 아니라 연동용 코드입니다.
